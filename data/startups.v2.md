# Startups v2 Dataset (Markdown)

## Purpose
Canonical startup dataset stored in Markdown table form.

## Columns
- `name`
- `country`
- `city`
- `region`
- `continent`
- `year_founded`
- `status`
- `specialty`
- `technology_modality`
- `technology_focus`
- `notable_breakthroughs`
- `funding_rounds`
- `website`
- `short_profile`
